#  Network Lab for Multiple Services

A comprehensive networking lab project designed using Cisco Packet Tracer to simulate and configure multiple network services in an enterprise environment. This project demonstrates practical implementation of routing, switching, VLANs, DHCP, network security, monitoring, and server-based services.

##  Features
- VLAN Configuration & Inter-VLAN Routing
- Static & Dynamic IP Addressing
- DHCP Server Configuration
- Router and Switch Configuration
- Wireless Network Setup
- Network Monitoring using SNMP
- Syslog & Remote Access Configuration
- Access Control Lists (ACLs)
- Basic Firewall & Security Concepts
- Client-to-Server Connectivity Testing
- Troubleshooting & Verification

##  Technologies & Tools Used
- Cisco Packet Tracer
- Routing & Switching
- IPv4 Addressing
- VLANs & Trunking
- DHCP
- SSH
- SNMP
- ACLs
- Wireless Networking

##  Project Objectives
- Simulate a real-world enterprise network environment
- Understand network communication between multiple services
- Configure secure and scalable network infrastructure
- Practice troubleshooting and monitoring techniques

## 🔍 Learning Outcomes
- Improved understanding of networking fundamentals
- Hands-on experience with Cisco devices and services
- Knowledge of network segmentation and security
- Practical exposure to enterprise-level networking concepts
